Dit is je gids voor alle basiskennis die je nodig hebt om je website zelf te beheren.

WordPress is het systeem achter je website. Het is als het brein van je site hier beheer je al je pagina's, afbeeldingen, content en meer. En het mooie is: je hoeft niets van programmeren af te weten. WordPress is het content management systeem (CMS) dat jouw website ondersteunt.

---

## WordPress Dashboard

Je WordPress dashboard is je kontrolepaneel. Dit is waar je alles beheert: je content, je instellingen, je plugins. Wanneer je inlogt, kom je hier terecht.

## Menu

Aan de linkerkant zie je een menu met allemaal opties. Dit zijn je gereedschappen. Hieronder leg ik kort uit wat elk onderdeel doet, zodat je weet waar je moet zijn als je iets wilt doen.

- Dashboard
- Bricks (Instellingen van de builder)
- Rank Math SEO (SEO-tool)
- Media (Hier staan al je afbeeldingen en bestanden)
- Berichten (Hier schrijf je artikelen en publiceer je content)
- Pagina's (Je website pagina's)
- Weergave
  - Thema's (De website builder dit wordt gebruikt)
  - Menu (Hier pas je de menu's van je website aan)
- Plugins (Alle extra tools en functies die je website nodig heeft)
- Gebruikers (Beheer je teamleden hier)
- Instellingen (De technische instellingen van je website)

Voelt overweldigend? Dat is normaal. Je hoeft niet alles te begrijpen. Veel van deze dingen zet je eenmalig in en sommige heb je nooit nodig.
